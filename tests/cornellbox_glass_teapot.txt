### Color Definitions ###
color red 0.75 0.1 0.1

### Shaders ###

phong_shader white_wall white white white 10

plane -6 0 0 1 0 0 red_wall

plane 0 -6 0 0 1 0 white_wall

plane 0 0 -6 0 0 1 blue_wall

# The Teapot 
# (Assuming it is centered at 0,0,0, we leave it there)
mesh ./obj/teapot.obj teapot_gold

# The Glass Sphere 
# Positioned at y = -5.2 (just above the floor at -6)
# and z = 3.0 (between the teapot and the camera)

### Lighting ###

# This will illuminate the face of the teapot and the sphere

### Camera ###
# Positioned at Z = 15 to capture the entire 12-unit wide room
